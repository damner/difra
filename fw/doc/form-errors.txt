Когда ajaxer отправляет формы, ошибки заполнения полей можно отображать несколькими методами.

Простое поле ввода:
<input name="inputname" type="text"/>
К input будет добавлен класс .problem.

Поле ввода в контейнере:
<div class="container">
	<input name="inputname" type="text">
</div>
При ошибке класс .problem будет добавляться к input и .container.

Если обязательное поле не заполнено и внутри .container есть .required, то к нему будет применен .fadeIn().

Если поле заполнено не правильно, внутри .container будет искаться .invalid. К нему будет применен .fadeIn(),
а если внутри .invalid есть элемент .invalid-text, в этот .invalid-text будет записан текст из параметра message.
